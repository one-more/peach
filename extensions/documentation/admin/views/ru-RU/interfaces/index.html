Интерфейсы
Данный интерфейс должны реализовывать шаблоны административной части.
admin_template_interface::get_info()
Метод должен возвращать массив с полями: 
                        [
                            'name',
                            'alias',
                            'preview' (необязательно. путь к изображению),
                            'icon' (необязательно. путь к изображению)
                        ]
Данный интерфейс должны реализовывать расширения daemon-extension
daemon_extension_interface::get_js()
Метод должен возвращать массив js файлов.
daemon_extension_interface::get_info()
Метод должен возвращать массив вида: 
                        [
                            'name'  => 'name',
                            'alias' => 'alias'
                        ]
daemon_extension_interface::get_version()
Метод должен возвращать строку с версией расширения
Данный интерфейс должны реализовывать редакторы
editor_interface::get_js()
Метод должен возвращать массив js файлов
editor_interface::get_css()
Метод должен возвращать массив css файлов
Данный интерфейс должны реализовывать расширения menu-extension
menu_extension_interface::create_layout()
Метод должен создавать отображение
menu_extension_interface::get_create_layout_html()
Метод должен возвращать html код для страницы создания отображения
menu_extension_interface::get_layout_params($id)
Метод должен возвращать параметры отображения
menu_extension_interface::get_page($link)
Метод должен возвращать массив вида: 
                        [
                            'class'         => 'class',
                            'controller'    => 'controller',
                            'position'      => 'position',
                            'id'            => 'layout id'
                        ]
Параметры:
$link - текущий url страницы
menu_extension_interface::get_urls()
Метод должен возвращать массив доступных ссылок
menu_extension_interface::get_version()
Метод должен возвращать строку с версией расширения
menu_extension_interface::get_info()\
Метод должен возвращать массив вида: 
                        [
                            'name'  => 'name',
                            'alias' => 'alias'
                        ]
menu_extension_interface::delete_extension_layouts($class)
Метод должен удалять все отображения расширения, переданного в параметре
Данный интерфейс должны реализовывать шаблоны сайта
site_template_interface::get_positions()
Метод должен возвращать массив позиций шаблона
site_template_interface::get_info()
Метод аналогичен методу в admin_template_interface
Родительский интерфейс
super_interface::start()
Точка входа в расширение
super_interface::delete()
Каждое расширение должно удалять себя самостоятельно. Это касается файлов расширения и его таблиц(если они есть)
Данный интерфейс должны реализовывать расширения user-extension.
user_extension_interface::get_info()
Метод аналогичен методу в daemon_extension_interface
user_extension_interface::get_version()
Метод аналогичен методу в daemon_extension_interface
Данный интерфейс должны реализовывать контроллеры видов
view_controller_interface::get_info()
Метод должен возвращать массив вида: 
                        [
                            'name'          => 'name',
                            'alias'         => 'alias',
                            'description'   => 'description'
                        ]
Данный интерфейс должны реализовывать контроллеры виджетов.
widget_controller_interface::get_info()
Метод должен возвращать массив вида: 
                        [
                            'name'  => 'name',
                            'alias' => 'alias'
                        ]
Данный интерфейс должны реализовывать расширения, у коотрых есть виджеты.
widget_extension_interface::get_widgets()
Метод должен возвращать массив вида: 
                        [
                            0 => ['name' => 'name', 'alias' => 'alias']
                        ]
widget_extension_interface::get_widget($class)
Метод должен подключать виджет, название которого передано в параметре
